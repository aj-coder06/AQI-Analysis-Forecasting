# AQI Analysis & Forecasting

A data analysis and forecasting project focused on studying Air Quality Index (AQI) trends across major Indian cities using time series analysis and machine learning forecasting models. The project performs preprocessing, visualization, and predictive forecasting on AQI datasets to analyze pollution patterns and future air quality trends.

---

## Project Overview

This project analyzes AQI data for multiple cities and applies forecasting techniques to predict future air quality levels. The workflow includes:

- Data preprocessing and cleaning
- AQI trend analysis
- Exploratory Data Analysis (EDA)
- Time series forecasting
- Model evaluation and comparison

The project demonstrates practical implementation of forecasting techniques using statistical and machine learning approaches on environmental datasets.
